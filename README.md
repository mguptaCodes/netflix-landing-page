# Netflix Landing Page Clone

## Project Overview

A responsive clone of the Netflix India landing page built using **HTML**, **CSS**, and **JavaScript**. This project replicates the look and feel of Netflix's homepage with a focus on responsive design, clean UI, and interactive components.


## Features

➜ Responsive design for desktop and mobile devices  
➜ Netflix-inspired landing page  
➜ Interactive FAQ accordion using JavaScript  
➜ Sign-In page UI  
➜ Clean and modern user interface 
➜ Custom CSS styling


## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/mguptaCodes/netflix-landing-page.git
```

### 2. Navigate to the project

```bash
cd netflix-landing-page
```

### 3. Open the project

Open the folder in **Visual Studio Code**.

### 4. Run the project

Open **index.html** in your browser

**OR**

Install the **Live Server** extension in VS Code and click **Go Live**.
